## Rs CSS

Tренажер для изучения css-селекторов. Тренажер состоит из нескольких игровых уровней. Каждый уровень включает в себя пример вёрстки и html-код, который отвечает текущему уровню. Некоторые элементы вёрстки выделены при помощи анимации. Задача пользователя - написать css-селектор, отвечающий всем выделенным элементам вёрстки. Прототип тренажера - приложение CSS Diner, из которого воспроизвена игровую часть.
